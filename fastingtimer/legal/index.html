Privacy Policy

Fasting Timer ("Service") can be used without signing in and does not collect or transmit any user information to external servers.

The Service processes the following data exclusively on-device and does not transmit it externally:

  • Fasting Data (Timer Records): Fasting start time, end time, duration, and completion status are stored on-device using SwiftData. No data is transmitted externally.
  • Health Data (Weight): Weight data is read and written via Apple HealthKit for tracking purposes. Data is stored only on the user's device (iPhone/Apple Watch).
  • Subscription Data: In-app purchase transactions are processed by Apple through StoreKit 2. The developer does not have access to payment information.
  • Notification Data: Local notifications are used to alert users when fasting windows complete. No push notification servers are used.

All data is processed exclusively on-device through Apple's SwiftData, HealthKit, and StoreKit frameworks. No data is transmitted to the developer or any third party.
